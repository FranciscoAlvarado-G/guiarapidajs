Sintaxis básica en JavaScript

Inicio | Tipos de datos | Sintaxis básica

La sintaxis en JavaScript, aunque ha evolucionado, siempre se ha mantenido sencilla y cercana al lenguaje humano en sus palabras clave. Aquí encontramos las estructuras fundamentales del lenguaje.

NOMBRE PALABRA RESERVADA SINTAXIS
Variable global var var x;
Variable bloque let let a;
Constante const const b = 5;
Condicional simple if if (condicion) { }
Condicional doble if...else if (condicion) { } else { }
Bucle for for for (let i = 0; i < 10; i++) { }
Bucle while while while (condicion) { }
Función function function nombre() { }
Retorno return return valor;
