INTRODUCCION

JavaScript es un lenguaje de programación interpretado, dinámico y orientado a objetos que se utiliza principalmente para crear interactividad en páginas web. Junto con HTML (estructura) y CSS (estilos), JavaScript permite desarrollar aplicaciones web completas, dinámicas y modernas. Fue creado en 1995 por Brendan Eich y hoy es uno de los lenguajes más utilizados en el mundo del desarrollo web.

Variables

Las variables permiten almacenar informacion para usarla mas adelante

En JavaScript se pueden declarar con:

  • let → Permite cambiar su valor.
  • const → Valor constante (no cambia).
  • var → forma antigua (ya casi no se usa).
